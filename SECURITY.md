# Security Policy

هذا المستودع بحثي ولا يقدم خدمة سريرية أو نظام قرار تنظيمي مباشرًا.

لا تنشر في Issues أو Pull Requests:

- بيانات مرضى أو معلومات صحية شخصية.
- مستندات سرية أو ملكية لشركات.
- مفاتيح API أو بيانات دخول.

للإبلاغ عن ثغرة أمنية، استخدم ميزة GitHub Private Vulnerability Reporting إن كانت مفعلة أو تواصل مع مالك المستودع بصورة خاصة.
