# Contributing

شكرًا لاهتمامك بالمشروع. الأولوية الحالية للمساهمات هي:

1. مراجعة دليل الترميز وتحسين تعريفات الفئات A–E.
2. التحكيم البشري للعيوب الذرية.
3. تحسين تصنيف `OTHER_SPECIFIC`.
4. مراجعة الرسائل التي لم تنتج مرشح A.
5. بناء معيار ذهبي للعلاقات عبر دورات المراجعة.
6. تطوير كود معالجة قابل لإعادة الإنتاج.

## الإبلاغ عن مشكلة

افتح Issue يتضمن:

- رقم الطلب وتاريخ الخطاب.
- اسم الورقة ومعرف السجل أو المرشح.
- النص المصدري ذي الصلة.
- التصنيف الحالي والتصنيف المقترح.
- سبب التعديل وفق دليل الترميز.

## مبادئ المساهمة

- لا تحذف النص المصدري أو تستبدله بتلخيص فقط.
- ميّز بين العيب والإجراء والتعليق غير المانع.
- لا تعتبر الرابط الزمني استمرارًا مثبتًا دون تحكيم.
- وثّق حالات الغموض بدل فرض تصنيف واثق.
