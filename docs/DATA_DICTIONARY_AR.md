# قاموس البيانات المختصر

## وحدات التحليل

| المصطلح | التعريف |
|---|---|
| الخطاب | وثيقة CRL مرتبطة بتاريخ ورقم طلب وفرع تنظيمي. |
| المقطع | جزء نصي مستخرج من قسم أو بند؛ قد يمثل عيبًا أو إجراءً أو تعليقًا أو نصًا روتينيًا. |
| المرشح | مقطع تنطبق عليه قواعد أولية ولم يعتمد نهائيًا بعد التحكيم البشري. |
| العيب الذري | أصغر مشكلة تنظيمية مستقلة يمكن حلها أو بقاؤها دون حل بمعزل عن المشكلات الأخرى. |
| الرابط الزمني المرشح | علاقة محتملة بين عنصر من دورة أقدم وعنصر من دورة أحدث. |

## أهم الحقول

| الحقل | المعنى |
|---|---|
| `candidate_id` | معرف فريد للمقطع المرشح. |
| `record_id` | معرف الخطاب الخام. |
| `normalized_application` | رقم وفرع الطلب بعد التطبيع. |
| `review_cycle` | ترتيب الخطاب زمنيًا داخل فرع الطلب نفسه. |
| `section_heading` | عنوان القسم الأصلي في الخطاب. |
| `source_excerpt` | النص المصدري الكامل للمقطع. |
| `target_class` | الفئة A أو B أو C أو D أو E. |
| `primary_domain` | المجال التنظيمي الرئيسي. |
| `reason_code` | رمز السبب التنظيمي. |
| `issue_statement` | صياغة مركزة للمشكلة، عند إمكان استخراجها. |
| `remediation` | الإجراء المطلوب لمعالجة المشكلة. |
| `quality_flags` | أعلام وصفية مثل الحجب وOCR والتجزئة والغموض. |
| `curation_quality` | مستوى وصفي تشغيلي، وليس احتمالًا إحصائيًا. |

## الفئات A–E

- **A:** مرشح عيب مؤثر في قابلية الموافقة.
- **B:** إجراء مطلوب دون مشكلة أم واضحة.
- **C:** تعليق غير مانع للموافقة.
- **D:** لغة روتينية أو قالب متكرر.
- **E:** غير قابل للحكم ويحتاج مراجعة.
